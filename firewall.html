Information paradox

Complementarity

Introduces 'stretched horizon'

The postulates

  1. Formation/Evaporation of black holes agrees with QM

  2. There is a set of semi-classical equations that work near the horizon

  3. An in-falling observer experiences nothing out of the ordinary ("No drama")

The Firewall Argument

[...] the purity of the Hawking radiation implies that the late radiation is fully entangled with the early radiation,

[while] the absence of drama for the infalling observer implies that it is fully entangled with the modes behind the horizon.

Why is this not allowed?

No-cloning theorem: There does not exist a quantum operation, $U_c$, which can be used to arbitrarily clone a quantum state.


Proof: Elementary result in linear algebra.

\[ U_c (a, 0) \rightarrow (a, a) \] so
\[ U_c (a + b, 0) \rightarrow (a + b, a + b) = (a,a) + (a,b) + (b,a) + (b,b) \]
but linear algebra says
\[ U_c (a + b, 0) = U_c (a,0) + U_c (b, 0) = (a,a) + (b,b) \]
